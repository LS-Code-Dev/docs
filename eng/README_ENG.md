## 📑 Project Index

### 1. [LS-OS (Development Environment Project)](ls-project.md)

*Design of a "Coding-Centric" operating system optimized for legacy machines.*

* **Status:** **INITIAL PLANNING** 🚀
* **Vision:** Create a smooth and lightweight development environment that allows coding on any hardware, eliminating distractions and slowdowns from commercial systems.
* **Core:** Kernel optimized for multitasking and a pre-configured development toolchain.
